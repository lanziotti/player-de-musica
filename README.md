# Player de Música

### Projeto em desenvolvimento no curso da Cubos Academy

#### Descrição:
Projeto de um player de música simples com suas principais funcionalidades (play, pause, stop, mudar de música).

#### Linguaguens utilizadas:
- HTML
- CSS
- Javascript
- ReactJS
